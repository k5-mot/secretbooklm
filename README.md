## Frontend

### 開発環境のセットアップ手順

```bash
npx create-miro-app@latest
# Name > miroapp-example
# Framework > React
# Flavor > Typescript
mv miroapp-example frontend
```

### 開発用サーバの起動手順

```bash
npm install
npm run build
# Run on localhost:3000
```
